# SugarBelly

SugarBelly is a SQL-backed global health analytics app that explores the relationship between sugar availability and obesity prevalence across countries.

The project combines food supply, health, and socioeconomic datasets to create interactive dashboards, country comparisons, risk segmentation, and machine-learning forecasts of future obesity trends.

## Project Goals

- Analyze sugar availability across countries and years.
- Compare sugar availability with obesity prevalence.
- Add socioeconomic context such as GDP, urbanization, and population.
- Build SQL analytics tables and views.
- Create an interactive Streamlit dashboard.
- Train a machine learning model to forecast future obesity trends.

## Planned Data Sources

- FAOSTAT Food Balance Sheets
- WHO Global Health Observatory
- World Bank Open Data
- Open Food Facts, optional later

## Tech Stack

- Python
- PostgreSQL
- SQL
- Streamlit
- Plotly
- scikit-learn
- XGBoost

## Important Note

This project studies country-level associations. It does not claim that sugar alone causes obesity. Obesity is influenced by many factors including diet, income, urbanization, physical activity, age structure, and healthcare context.
